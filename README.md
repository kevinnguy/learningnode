### First node project

Used `yo angular-fullstack` to generate project

### Instructions

Before running `yo angular-fullstack`, run:

`
npm install -g yo
npm install -g generator-angular-fullstack
npm install -g bower-cli
npm install -g bower
brew install mongodb
`
